# custom_targets
Custom target definitions for Mbed5

Currently supported:
- STM32F407VE_BLACK
- BLUEPILL_F103RB
  same as BLUEPILL_F103C8, but uses 128 kB flash and newlib-nano for bare_metal profile
- WEACT_F411CE
  WeACT MiniF4 F411CE
- LPC812 and LPC824 
  from Mbed-os-5.15, targets were removed in Mbed6
